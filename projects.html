Breast Cancer Detection

Implemented a convolutional neural network (CNN), a support vector machine (SVM), and a vision transformer (ViT) model to detect breast cancer in mammogram x-ray images. Applied data pre-processing techniques to improve model accuracy, achieving an average accuracy of at least 70% and peak accuracies exceeding 80%.

Laser Chase

Developed an interactive, embedded laser pointer toy for cats using Arduino and Raspberry Pi, featuring YOLOv8-based Python object detection to autonomously track and engage with pets. Designed and 3D-printed a custom enclosure with both automated and manual control modes, allowing user-operated precision via joystick, enhancing product usability and functionality.
