Laser Chase Cat Toy

This project is an interactive, embedded laser pointer toy for cats. The goal of this project was to create a device that could entertain my cat when I am unable to play with him. Thus, Laser Chase was created.

This toy features two modes: an interactive mode, where users can control the laser pointer using a joystick, and an automatic mode, which moves the laser around autonomously. The automatic mode is equipped with computer vision technology to detect the cat's location and move the laser away from them.

To develop this device, I used an Arduino Mega to process inputs from the power button, mode button, and joystick. The Arduino controls the laser pointer and the pan-tilt mount.

A Raspberry Pi 4 was used for the computer vision component. This feature employs the YOLOv8 model for object detection and sends the cat's location to the Arduino via a serial connection.

The case for this project was custom-designed and 3D-printed using CAD software. Additionally, some parts were laser-cut using custom vector images. Designing the parts involved an iterative process as sometimes the designs would not work with the parts I was using.
